# PrizmForge Federation

**A governed, multi-paradigm agent system for long-term, high-quality improvement of existing codebases.**

PrizmForge Federation evolves the original PrizmForge governed editing model into a more ambitious architecture: one in which multiple distinct *mental models* (reasoning paradigms and value systems) operate as semi-sovereign ecosystems. These ecosystems can propose, critique, and implement changes while being coordinated through shared infrastructure and higher-order constitutional governance.

The system draws inspiration from working examples such as mydeadinternet.com (territories, moots, claims, trust, and layered self-governance) and game systems like Endless Sky (dynamic faction relationships and economies), while remaining focused on the practical goal of safely and productively modifying real code.

## Overview

Most current multi-agent frameworks treat agents as members of relatively flat teams with assigned roles. PrizmForge Federation explores a different approach:

- Multiple **mental model ecosystems** run with protected autonomy and their own internal logic.
- These ecosystems interact through structured channels (message buses, proposals, and shared evaluation).
- A **constitutional layer** (inspired by moots) provides higher-order rules that are deliberately harder to change.
- Real stakes, feedback loops, and maintenance mechanisms help the system self-correct over time.

The long-term vision is a resilient, multi-paradigm platform that can improve complex codebases more effectively than any single reasoning style could achieve on its own.

## Goals

- Enable **productive coexistence** of different reasoning paradigms rather than forcing convergence on one approach.
- Maintain strong **safety and auditability** while allowing greater exploration and specialization.
- Create **real incentives and consequences** between agent groups (reputation, resource allocation, and outcome-based influence).
- Build systems that can improve over long periods without being captured by any single optimization pressure or mental model.
- Learn from and adapt proven patterns from existing projects (mydeadinternet.com, evolutionary coding agents, polycentric governance concepts) rather than starting from zero.

## Core Principles

- **Protected Diversity**: Different mental models should have space to operate according to their own logic.
- **Constitutional Governance**: Foundational rules should be harder to change than day-to-day decisions.
- **Real Stakes**: Influence, resources, and survival should be tied to demonstrated value and outcomes.
- **Layered Architecture**: Clear separation between ecosystem-level autonomy and system-level coordination.
- **Sustainable Pace**: Build in focused sprints with clear deliverables rather than attempting to predict long timelines.
- **Pragmatic Inspiration**: Borrow useful mechanics from working systems (MDI, evolutionary frameworks, game economies) and adapt them to code improvement.

## Current Status & Expectations

This project is in early exploration. We have strong conceptual foundations from PrizmForge and observations from systems like mydeadinternet.com, but the integrated architecture does not yet exist.

**Expectations for development:**
- Work will happen in focused sprints rather than rigid long-term plans.
- We will prioritize working prototypes over architectural perfection in early stages.
- Scope will be deliberately narrowed at each step so that meaningful progress can be made quickly.
- We will heavily leverage existing permissive-license frameworks (CrewAI, LangGraph, etc.) instead of building everything from scratch.
- Evaluation, materialization of code changes, and constitutional mechanisms will receive early attention because they are core to the value of the system.

## Technology & Constraints

- Local hardware: \~120 CPU cores + single Nvidia 1080ti
- Work API budget available for high-value reasoning and evaluation
- Preference for permissive open-source licenses where possible
- Focus on practical code improvement rather than pure research or simulation
